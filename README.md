# ImageProcessingAPI

Image Processing API using express , typeScript , Sharp , and testing with Jasmine

---

## how to use

- to install required modules
  `npm install`
- to build app 
   `npm run build`
- you can use this to build app & run server
  `npm run dev`
- to test app 
  `npm run test`
- lint
   `npm run lint`
- prettier
    `npm run prettier`
